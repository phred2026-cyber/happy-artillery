# Changelog

## [1.0.0] - 2026-03-01

### Added
- Initial public release
- Fireball shooting from Happy Ghasts (right-click with Fire Charge while riding)
- Ghast Cry ability (right-click with Ghast Tear while riding)
- Ammo system with passive regeneration (default: 200 max, 1 per 5 minutes)
- Heat and overheat mechanics (overheating triggers a sphere explosion)
- Biome-based heat behavior (Hot, Cold, Nether, Base)
- Water cooling mechanic (submerged ghasts cool but can't fire)
- Full JSON config file at `config/happy-artillery.json`
- Control item system (fire charge + ghast tear are tagged as control sticks)
